# Ajedrez

_Ajedrez: Chess in Spanish_

## Starting the Project

---

### First Boot Up

```sh
# Compile the TypeScript codebase -- this needs to be done after any changes to the source files.
$ yarn build
```

### Starting the server

```sh
$ yarn start
```

## Testing & Type Checking

---

### Type Checking the Project

```sh
$ yarn tc
```
